# Веб-приложение для ведения заметок Notelib

Курсовая работа выполнена с использованием:
+ HTML
+ CSS
+ Javascript

Для Backend части использовались:
+ NodeJS + Express

Реализованный функционал:
+ Создание заметок
+ Удаление заметок
+ Активность заметок
+ Просмотр всех существующих заметок
+ Базовая валидация при попытке создания пустой заметки, а так же выведение понятных ошибок при неудачной загрузке выводимой информации

Хранение данных заметок было упрощено и реализовано в файле database.js
